# create-prediction-market-agent
Scaffold a prediction market agent project. LangChain, OpenAI Agents, or vanilla TypeScript.

```bash
npx create-prediction-market-agent my-agent
npx create-prediction-market-agent my-agent --framework openai
npx create-prediction-market-agent my-agent --framework langchain
```

## Frameworks
- **vanilla** — Pure TypeScript with `prediction-market-context`
- **openai** — OpenAI function calling with `openai-agents-prediction-markets`
- **langchain** — LangChain ReAct agent with `langchain-prediction-markets`
